🎨 MatForum Frontend Dokumentacija

Moderni React aplikacija sa TypeScript-om i TanStack Router-om

📋 Pregled

MatForum frontend je moderna web aplikacija izgrađena sa React-om i TypeScript-om. Koristi najnovije tehnologije i best practices za izgradnju performantnih i održivih aplikacija.

Type-safe routing za React

📁 Struktura Projekta

src/ ├── components/ # Reusable komponente │ ├── features/ # Feature-specific komponente │ │ ├── questions/ # Komponente za pitanja │ │ ├── voting/ # Komponente za glasanje │ │ └── auth/ # Komponente za autentifikaciju │ └── ui/ # Osnovne UI komponente ├── routes/ # TanStack Router rute │ ├── ~(public)/ # Javne rute │ │ └── ~_auth/ # Auth rute (login, signup) │ └── ~_protected/ # Zaštićene rute │ ├── ~ask/ # Postavi pitanje │ └── ~questions/ # Lista pitanja ├── lib/ # Utility funkcije ├── hooks/ # Custom React hooks └── api/ # API klijenti i tipovi

🛠️ Instalacija i Pokretanje

Instalacija zavisnosti

cd MatfForum/src/frontend
npm install --legacy-peer-deps

Pokretanje development servera

npm run dev

Aplikacija će biti dostupna na http://localhost:5173

Build za produkciju

npm run build

Preview produkcijskog build-a

npm run preview

Autentifikacija

  • Registracija korisnika
  • Prijava (login)
  • Odjava (logout)
  • JWT token management

Pitanja

  • Pregled svih pitanja
  • Postavljanje novih pitanja
  • Pretraga pitanja
  • Filtriranje po kategorijama

Odgovori

  • Pregled odgovora na pitanja
  • Dodavanje novih odgovora
  • Uređivanje odgovora

Glasanje

  • Upvote/Downvote za pitanja
  • Upvote/Downvote za odgovore
  • Prikaz broja glasova

🔌 API Integracija

Frontend komunicira sa backend-om preko API Gateway-a na http://localhost:5000

Glavni API Endpoints

  • POST /api/auth/login - Prijava korisnika
  • POST /api/auth/register - Registracija korisnika
  • GET /api/questions - Lista pitanja
  • POST /api/questions - Kreiranje pitanja
  • GET /api/questions/{id}/answers - Odgovori na pitanje
  • POST /api/answers - Kreiranje odgovora
  • POST /api/votes - Glasanje

🎨 Styling

Projekat koristi Tailwind CSS za styling sa dodatnim custom komponentama iz Radix UI.
